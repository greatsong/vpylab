## Lab 1: Teachable Machine (내가 가르치는 AI)

### 기능 테스트
- [ ] 웹캠 스트림 정상 표시
- [ ] 3개 클래스 라벨 입력 가능
- [ ] 각 클래스별 샘플 수집 (버튼 누르는 동안 캡처)
- [ ] 수집된 샘플 수 표시
- [ ] "학습 시작" 버튼 → 모델 학습 진행
- [ ] 학습 진행률(에폭) 표시

- [ ] 예측 결과에 따라 3D 오브젝트 색상 변경
- [ ] 초기화 버튼으로 전체 리셋

### 에러 처리
- [ ] 카메라 권한 거부 시 안내 메시지
- [ ] 샘플 부족 시 학습 방지 (최소 5개/클래스)

## Lab 2: RL Playground (AI 걸음마 놀이터)

### 기능 테스트
- [ ] 5×5 그리드 정상 렌더링
- [ ] 에이전트(시작) / 목표(도착) 위치 표시
- [ ] 장애물 배치 (클릭으로 토글)
- [ ] "학습 시작" → Q-Learning 에피소드 진행
- [ ] Q-값 히트맵 시각화 (색상 그라데이션)
- [ ] 에이전트 이동 애니메이션

- [ ] 에피소드 수 / 누적 보상 그래프
- [ ] 보상 함수 파라미터 조절 (목표 도달 보상, 이동 패널티)
- [ ] 초기화 버튼

### 에러 처리
- [ ] 목표 도달 불가능한 맵 감지
- [ ] 무한 루프 방지 (최대 스텝 제한)

## Lab 3: Neural Network 3D Viz (뉴런이 살아 움직인다)

### 기능 테스트
- [ ] XOR 문제 데이터 시각화 (4개 점)
- [ ] 신경망 구조 시각화 (입력 2 → 은닉 4 → 출력 1)
- [ ] 뉴런 노드를 원으로 표시
- [ ] 가중치를 연결선으로 표시 (굵기 = |가중치|, 색상 = ±)
- [ ] "학습 시작" → 에폭별 학습 진행
- [ ] 활성화 값 → 노드 발광 강도
- [ ] Loss 그래프 실시간 업데이트
- [ ] Decision boundary 시각화 (2D 컬러맵)
- [ ] 에폭 슬라이더로 특정 시점 탐색
- [ ] 학습률 조절

### 에러 처리
- [ ] NaN 가중치 감지 → 자동 리셋 제안
- [ ] 학습 미수렴 시 안내 메시지
