# 행동 강령 (Code of Conduct)

## 우리의 서약

VPy Lab은 **학생과 교사가 함께 만드는 오픈소스 교육 플랫폼**입니다.
나이, 성별, 국적, 경험 수준에 관계없이 모든 참여자가 안전하고 존중받는 환경을 만들겠습니다.

## 미성년자 안전 정책

VPy Lab의 주요 사용자와 기여자에는 **고등학생(15~18세)**이 포함됩니다.
다음 규칙은 모든 참여자에게 적용됩니다:

### 필수 규칙

1. **개인정보 보호**: GitHub Issues, PR, Discussions에서 학생의 실명, 학교명, 학년/반, 사진 등 개인 식별 정보를 게시하지 않습니다
2. **1:1 비공개 대화 금지 권고**: 성인 기여자가 미성년 기여자에게 GitHub 외부 채널(개인 SNS, 이메일 등)로 1:1 연락하는 것을 금지합니다
3. **교육 콘텐츠 기준**: 미션 콘텐츠에 폭력적, 성적, 차별적 내용을 포함할 수 없습니다
4. **존중과 배려**: 코드 리뷰는 학습의 기회입니다. 비난이 아닌 건설적 피드백을 제공합니다

### 신고 채널

행동 강령 위반을 발견하면 다음으로 신고해주세요:
- **이메일**: vpylab.conduct@gmail.com
- **GitHub**: Issue에 `[CoC]` 태그를 붙여 신고

신고는 24시간 내 확인 응답, 7일 내 판단을 목표로 합니다.

### 집행

- 현재는 프로젝트 창시자가 CoC를 집행합니다
- 교사 기여자가 확보되면 2인 이상 CoC 위원회로 확대할 예정입니다
- 학생이 가해자인 경우: 경고 → 기여 제한 → 차단 (3단계)
- 성인이 학생에 대한 가해자인 경우: 즉시 차단

## 일반 행동 강령

이 프로젝트는 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)을 채택합니다.

### 기대하는 행동

- 환영하는 언어 사용
- 다양한 관점과 경험 존중
- 건설적 비판의 수용
- 커뮤니티를 위한 최선의 판단

### 용납하지 않는 행동

- 성적 언어/이미지 사용, 부적절한 관심
- 트롤링, 모욕적 발언, 인신공격
- 공개/비공개 괴롭힘
- 동의 없는 개인정보 공개

---

*이 행동 강령은 VPy Lab 프로젝트의 모든 공간(GitHub, 배포 사이트, 관련 소통 채널)에 적용됩니다.*
